The Inference Chip Architecture Map: 12 Companies Sorted by Three Bottlenecks
Damnang 2026-07-01 15:55 Asia/Taipei
  • 作者把 12 家推理晶片公司放進三個瓶頸框架：記憶體頻寬、彈性成本、部署 / 功耗。

  • 作者認為極端硬編碼不再是終局，市場正往「保留部分可重配置性」的中間地帶收斂。
  • NVIDIA 的核心防守不是單顆 GPU，而是整個 rack/system：Rubin GPU、Vera CPU、NVLink 6、網路與安全一起打包。
  • 真正難打的護城河不是算力規格，而是 CUDA 與既有軟體生態的切換成本。
  • Google TPU、Amazon Trainium、Microsoft Maia、Meta MTIA 會從內部工作負載端，持續吃掉外部推理晶片的可服務市場。
  • 作者最後把可投資重點拉回公共基礎層：foundry wafers、memory、packaging、power、rack，而不是去猜哪一家私有新創會贏。

ECTC 2026: The Three HBM Battlegrounds Defining the Next AI Hardware Race
SEMIVISION 2026-07-01 11:19 Asia/Taipei
  • 作者把 HBM 從 HBM2 / 2E / 3 / 3E 往 HBM4 / 4E 的演進，整理成同一條路線：更高 per-pin data rate、更多 I/O、更多堆疊層數、更大容量、與邏輯 die 更短的距離。
  • 但更高帶寬不是免費午餐：I/O 變密，封裝 routing 變更難，signal integrity 壓力也更高。
  • 堆疊變高會讓 power delivery 與 thermal management 複雜度同步上升。
  • 封裝尺寸變大後，warpage、reliability、yield 都會變成新的瓶頸。
  • 作者明講下一階段不是單純 memory design，而是 system engineering 問題。
  • 公開預覽列出的第一個技術戰場是 Intel 的 EMIB-T，目標是支撐 12+ Gb/s 的 HBM4E 互連。
  • 第二與第三個戰場分別是 KAIST 的 3D-HI-HBM 電源傳輸，以及 Samsung 把 interconnect 變成熱路徑一部分的 HCB HBM。
  • 最後作者把投資鉤子拉到供應鏈與 AMD：HBM 供應吃緊時，LPDDR 替代路徑會變成另一個觀察點。

5. SEMIVISION｜ECTC 2026: The Three HBM Battlegrounds Defining the Next AI Hardware Race
- 作者：SEMIVISION
- 來源：Substack 文章
- 標題：ECTC 2026: The Three HBM Battlegrounds Defining the Next AI Hardware Race
- 發布時間：2026-07-01 11:19 Asia/Taipei
- 來源狀態：付費牆/僅公開預覽
- 連結：https://tspasemiconductor.substack.com/p/ectc-2026-the-three-hbm-battlegrounds
- 立場：中性
- 時間框架：中長期
- 重點 1：作者把 HBM 從 HBM2 / 2E / 3 / 3E 往 HBM4 / 4E 的演進，整理成同一條路線：更高 per-pin data rate、更多 I/O、更多堆疊層數、更大容量、與邏輯 die 更短的距離。
- 重點 2：但更高帶寬不是免費午餐：I/O 變密，封裝 routing 變更難，signal integrity 壓力也更高。
- 重點 3：堆疊變高會讓 power delivery 與 thermal management 複雜度同步上升。
- 重點 4：封裝尺寸變大後，warpage、reliability、yield 都會變成新的瓶頸。
- 重點 5：作者明講下一階段不是單純 memory design，而是 system engineering 問題。
- 重點 6：公開預覽列出的第一個技術戰場是 Intel 的 EMIB-T，目標是支撐 12+ Gb/s 的 HBM4E 互連。
- 重點 7：第二與第三個戰場分別是 KAIST 的 3D-HI-HBM 電源傳輸，以及 Samsung 把 interconnect 變成熱路徑一部分的 HCB HBM。
- 重點 8：最後作者把投資鉤子拉到供應鏈與 AMD：HBM 供應吃緊時，LPDDR 替代路徑會變成另一個觀察點。
- 明確提到的標的：INTC, AMD
- 明確提到的族群：HBM、先進封裝、記憶體、AI 硬體
- 催化因素：ECTC 2026 技術揭露、HBM4E、LPDDR 替代路徑
- 操作建議：觀望
- 風險或保留條件：全文仍在付費牆後，公開預覽只夠看見技術戰場，還看不到作者完整點名的受益順序。
- 我是否需要親自讀原文：需要；需要，因為真正的供應鏈排序與受益強弱仍在付費牆後。

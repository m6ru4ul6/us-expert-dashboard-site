The Memory Crisis: Can CXL Be the Answer?
Damnang 2026-07-06 10:55 Asia/Taipei
  • 作者指出 hyperscalers 的記憶體 capex 占比從 2023-2024 年約 8% 估計升到 2026 年約 30%，記憶體從配角變成 AI 基建成本核心。
  • CXL 兩三年前曾被視為 AI 時代已死，但 2026 年因記憶體短缺與推理工作負載變化重新被市場重估。
  • AI 推理的 KV cache 讓容量和頻寬同等重要，CXL 的價值在於把較少使用的記憶資料放到擴充記憶體。
  • Meta 的 Vistara 案例把淘汰伺服器的 DDR4 透過 CXL 重新部署到生產資料中心，顯示記憶體昂貴時 reuse / pooling 有經濟性。
  • Microsoft Azure M-series CXL-attached memory preview 與 Meta 論文讓 CXL 從供應商推動轉為需求端驗證。
  • 對投資而言，CXL 不必立即取代 HBM；更像是在推理容量瓶頸上提供分層解法，需追蹤 Intel/Arm 平台支援與 hyperscaler 採用。
